## About The Project

[![Product Name Screen Shot][product-screenshot]](https://example.com)

"TUNE MADNESS" is a MERN stack pet project aimed at making the task of finding doubles for DJs more manageable. By considering a wide range of factors, DJs can more easily mix songs, enhancing the overall experience for both DJs and listeners.

**Our vision**

To create a web application that simplifies the process of finding doubles for DJs, making it easier for them to mix songs and for listeners to enjoy a seamless musical experience.

### Built With

* [MongoDB](https://www.mongodb.com/)
* [Express.js](https://expressjs.com/)
* [React.js](https://reactjs.org/)
* [Node.js](https://nodejs.org/)
